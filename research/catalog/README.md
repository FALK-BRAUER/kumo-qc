# research/catalog/

The master enumeration layer — every experiment + variant, with its verdict, in one place.

- **Holds:** `experiment_catalog.csv` (the experiment ledger), `variant-catalog.md` (the phase-kind → existing-variants → tunable-knobs map folded from ADR-0001 §6).
- **Goes here:** enumerations / indexes of experiments + variants — the "what have we tried" tables.
- **Does NOT:** per-experiment writeups (→ `../experiments/`), the numeric result ledger (→ `results/bt-results.csv`), or runnable code.
