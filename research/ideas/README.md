# research/ideas/

Idea banks, candidate strategies, external-source analyses — the not-yet-tested pile.

- **Holds:** proposed strategies / models and analyses of others' approaches (e.g. `qc_strategy_analysis_bct_ideas.md`, `risk_sizing_model.md`).
- **Goes here:** forward-looking proposals and "what could we try" notes, before they become experiments.
- **Does NOT:** experiments already run (→ `../experiments/`), validation methodology (→ `../methodology/`), or private-source feed logs (→ `../sources/`).
