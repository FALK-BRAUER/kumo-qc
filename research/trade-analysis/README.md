# research/trade-analysis/

Trade-behavior studies — the #174 mandate.

- **Holds:** analyses of the actual trades a run produced (winners/losers, hold times, payoff distribution, entry/exit quality), e.g. `e40c_trade_analysis.md`.
- **Goes here:** "why did the P&L come out this way" trade-level forensics.
- **Does NOT:** the experiment writeup itself (→ `../experiments/`), cloud/local parity diffs (→ `../parity/`), or the numeric ledger (→ `results/`).
