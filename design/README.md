# design/

Design documents and architecture sketches.

- **What's here:** Empty placeholder — reserved for future architecture decision records (ADRs), wireframes, and methodology mapping documents that are not yet formalized in `docs/` or `research/`.
- **What goes in:** Pre-ADR sketches, whiteboard captures, and exploratory design notes.
- **What does NOT go here:** Committed ADRs (use `docs/`), backtest analysis (use `research/`), or implementation code.
