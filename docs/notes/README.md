# docs/notes/

Ephemeral / historical docs — point-in-time notes that are NOT the living charter.

- **Holds:** dated triage notes, support tickets, one-off design specs, reproduction plans, historical analyses (e.g. `backlog-triage-2026-05-30.md`, `qc-support-ticket-compile-cache.md`, `worker-dispatch-protocol.md`, `GH25_intraday_design_spec.md`).
- **Goes here:** anything that records a moment in time and won't be continuously maintained.
- **Does NOT:** the governing/living charter — `ARCHITECTURE.md`, `PHASES.md`, `ACCEPTANCE_CONTRACT.md` (stay in `docs/`); the rules (`CONVENTIONS.md`) live at repo root.
