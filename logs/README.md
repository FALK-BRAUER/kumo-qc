# logs/

Runtime log output — not committed.

- **What's here:** Empty or transient log files from local backtest runs, performance window audits, and ad-hoc debugging sessions.
- **What goes in:** `.log` files emitted by running strategies or scripts. These are gitignored.
- **What does NOT go here:** Structured result artifacts (use `results/`), committed documentation, or handoff notes (use `zz_handoffs/`).
