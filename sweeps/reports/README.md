# sweeps/reports/

Aggregated sweep leaderboards — one row per variant, sorted.

- **Holds:** `<sweep-id>.csv` / `.md` — per-variant Sharpe/Ret/DD/orders + windows + config-hash + marker.
- **Curated** leaderboards may be tracked; raw bulk output is gitignored.
- **Promotion:** sweep winners graduate into `results/bt-results.csv` (the master ledger) with full provenance.
- **Does NOT:** be the master ledger (that's `results/`).
