# lambdamart_scanner_ranker

Opt-in ranking phase that applies an exported BCT/George-style scanner artifact to live signal candidates.
It supports the original LambdaMART/tree artifact and the #467 linear opportunity-ranker artifact.
It belongs after `signal` and before entry selection/sizing in experimental scanner-alignment configs.
Do not put George OCR/watchlist labels, trained research panels, or backtest output in this directory.
